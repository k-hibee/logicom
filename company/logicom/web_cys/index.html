WEB ORDER SYSTEM

配車システムと連動し、配送のご依頼に効率的に対応。
24時間365日、どこからでもアクセス可能です。

『WEB受注システム』は、当社の配車システムと連動したシステムで、お客様の配送依頼をスムーズかつ確実に処理します。
なお、ご利用にあたっては、事前にお客様専用のID、パスワードが必要になります。
ID、パスワードの発行には一切、費用のご負担は必要ありません。
詳しくは、当社最寄りの営業所または担当者までご連絡ください。

システムの流れ

ご利用のメリット

devices

24時間どこからでも

インターネット環境があればPCはもちろん、スマートフォンからでもOK。オフィスやご家庭、出先など、どこからでもご依頼や配送状況の確認が24時間可能です。

trending_down

効率化とコスト削減

電話やFAXでやり取りする時間（効率化）と経費（人件費や通信料）が節約できます。ペーパーレス化にも貢献します。

analytics

管理資料として利用

配送のデータ（履歴、距離、料金など）をシステム上で確認・出力できますので、お客様の社内管理業務にそのままご利用いただけます。

check_circle

発注ミスの軽減

お客様ご自身による入力で申し送りミスが軽減できます。また、受注確認や配送完了報告のご案内により二重チェックを行っています。

sentiment_satisfied_alt

ご利用はもちろん無料

運送経費以外、システム利用料などのコストをお客様にご負担いただくことはありません。（※インターネット通信にかかる費用は除きます）

システムのご利用にあたって

schedule 受付時間について
24時間承っています。
ただし、午後6：00以降のご依頼で、翌日の午前9：00以前をご希望される場合は、お手数ですが担当者にご確認ください。
inventory_2 複数の荷物をご依頼される場合
主なお荷物を「ご依頼品欄」にご入力いただき、その他のお荷物については「備考欄」にご入力ください。
local_shipping 複数の車両をご依頼される場合
車種欄よりご希望の車種をお選びいただき、その下に台数をご入力ください。
help_outline 車種・台数が不明な場合
お荷物の量や種類に応じて、当社で最適な車種と台数を選択させていただきますのでご安心ください。
contact_phone ご連絡先について
ご担当者様に確実に連絡をお取りできる電話番号（携帯電話など）をご入力ください。
timer ご希望の配送日時について
天候や道路事情等により、ご希望の時刻に到着できないことがあります。
その際には、当社より事前にご連絡し、時刻変更のご相談をさせていただきます。
manage_search 配送状況・お問い合わせ
配送状況については、システム内の「配送状況確認ページ」においていつでもご確認いただけます。
その他ご不明な点やご質問などがございましたら、当社までお問い合わせください。
